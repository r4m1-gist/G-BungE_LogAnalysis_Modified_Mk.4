G-BungE Log Analysis Web

기존 Python CLI 분석기를 브라우저 폼으로 실행하는 로컬 웹 러너입니다.

실행 방법

GitHub Pages에서는 Python 분석 코드를 실행할 수 없습니다. 실제 로그 분석은 로컬에서 아래 명령으로 서버를 켠 뒤 접속합니다. 

python3 web/server.py

서버가 켜지면 아래 주소를 엽니다.

http://127.0.0.1:8765

웹 러너에서 하는 일

  • 로그 그룹과 `.log` 파일을 선택합니다.
  • GPS, torque, thermal, power 분석 플롯을 고릅니다.
  • Matplotlib 결과를 PNG로 저장하고 브라우저에 표시합니다.
